# Project README

This project uses AgentCompanyOS company engineering standards.

## Development

- Read `AGENTS.md` before making changes.
- Keep changes focused and tested.
- Use migrations for database changes.
- Do not hardcode secrets.

## Validation

```bash
make test
```

If no `Makefile` exists, use the project-native test command and document it here.

## Risk

High-risk actions require approval before execution.
