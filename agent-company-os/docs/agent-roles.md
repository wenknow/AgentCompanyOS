# Agent Roles

- `chief_of_staff`: routes Founder intent, creates tasks, summarizes status, and requests approvals.
- `product`: drafts PRDs, user stories, acceptance criteria, and roadmap notes.
- `cto`: drafts technical plans, architecture breakdowns, and risk reviews.
- `backend`: drafts API, database, test, and Codex prompt plans.
- `content`: drafts content only; it never publishes.
- `compliance`: reviews risk, claims, investment language, and revision needs.
